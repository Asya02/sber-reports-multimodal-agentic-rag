# Ключевая информация из изображения

## Заголовок
**Решения Сбера в области ИИ, разработанные в 2023 году**

## Программные решения

### 1. ESGify
- **Описание**: Библиотека для классификации ESG-новостей.
- **Цель**: Оценка ESG-рисков компании и соответствие контрагентов ESG-требованиям.
- **Принципы работы**: Обработка текстовых данных и анализ новостей для выявления ESG-рисков.
- **Преимущества**: Решает проблему отсутствия стандартизированных ESG-данных и может распознавать тексты по 47 ESG-темам.

### 2. ASCAD Light
- **Описание**: Модель для повышения качества солнечных панелей на крупносерийном производстве.
- **Цель**: Улучшение качества солнечных панелей.
- **Принципы работы**: Выявление аномалий и поиск причин образования дефектов.
- **Преимущества**: Позволяет выявлять дефекты и класс качества без предварительного формирования данных.

### 3. Eco4cast
- **Описание**: Библиотека для снижения выбросов парниковых газов при обучении нейросетей.
- **Цель**: Снижение выбросов парниковых газов.
- **Принципы работы**: Анализ углеродной стоимости электроэнергии и выбор оптимальных временных интервалов.
- **Преимущества**: Позволяет сокращать выбросы в среднем на 25%.

### 4. Eco2AI
- **Описание**: Библиотека для мониторинга углеродного следа моделей машинного обучения.
- **Цель**: Контроль энергетического потребления и углеродного следа.
- **Принципы работы**: Оценка углеродного следа в процессе выполнения кода.
- **Преимущества**: Позволяет вести точный учет возможностей для отдельных вычислительных процессов.

### 5. WaRP
- **Описание**: Датасет и модель компьютерного зрения для сортировки отходов.
- **Цель**: Улучшение сортировки на заводах по переработке отходов.
- **Принципы работы**: Анализ отходов на конвейере с использованием баз данных изображений.
- **Преимущества**: Позволяет распознавать объекты, которые могут сильно деформироваться.

## Общая информация
- Все решения направлены на улучшение экологической устойчивости и эффективности производственных процессов через использование технологий искусственного интеллекта.
