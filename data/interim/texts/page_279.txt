# Предоставление равного доступа к финансовым услугам

## Основная информация

- **Организация:** Сбер
- **Цель:** Создание доступной и удобной среды для всех клиентов, включая людей с ограниченными возможностями.
- **Инициатива:** «Особенный банк» — адаптация банковских услуг для клиентов с особыми потребностями.

## Ключевые данные

- **Количество точек обслуживания:** 5,8 тыс. (адаптированы для людей с инвалидностью).
- **Категории посетителей:**
  - Нераспределенные и слабовидящие клиенты
  - Люди с нарушениями слуха и речи
  - Маломобильные люди
  - Люди с ментальными особенностями

## Услуги банка

- **Услуги для слабовидящих:** 
  - Специальные банкоматы с тактильными метками.
  - Голосовое сопровождение операций.
  
- **Услуги для людей с нарушениями слуха:**
  - Бесплатные услуги переводчика жестового языка (РЖЯ).
  - Доступ к мобильному приложению для видеозвонков с переводчиком.

- **Услуги для маломобильных людей:**
  - Адаптированные точки обслуживания.
  - Возможность получения услуг на дому через приложение Сбербанк Онлайн.

## Примечания

- В 2023 году банк планирует расширить доступность видеозвонков на русском жестовом языке.
- Большинство услуг доступны через мобильное приложение Сбербанк Онлайн, что позволяет клиентам получать обслуживание без необходимости посещения офиса. 

## Вывод

Сбер активно работает над улучшением доступности финансовых услуг для всех категорий клиентов, особенно для людей с ограниченными возможностями, внедряя современные технологии и адаптируя существующие сервисы.
