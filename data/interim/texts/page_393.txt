# Описание ключевой информации на изображении

## Направление: Риск-менеджмент
- **Элемент отчетности**: 
  - Описание процессов организации по выявлению и оценке рисков, связанных с климатом.
  - Описание процессов организации по управлению рисками, связанными с климатом.
  - Описание того, как процессы выявления, оценки и управления рисками, связанными с климатом, интегрированы в общий процесс организации.
  - Описание исходных данных, применяемых в процессе идентификации климатических рисков, изменениях в процедурах идентификации, анализа, мониторинга рисков по сравнению с предыдущим годом.
  - Описание процессов организации по выявлению, анализу, мониторингу и приоритизации возможностей, связанных с климатом.

## Показатели и цели
- **Элемент отчетности**:
  - Сведения о показателях, используемых организацией для оценки рисков и возможностей, связанных с климатом, в соответствии с ее стратегией и процессом управления рисками.
  - Сведения об объеме выбросов парниковых газов по Охвату 1, Охвату 2 и 3, где применимо.
  - Описание целей, используемых организацией для управления рисками и возможностями, связанными с климатом, и фактических результатов по сравнению с целевыми показателями.
  - Сведения об отраслевых показателях в области климата, относящихся к деятельности организации, выбросах.
  - Сведения об инвестициях, направленных на управление ключевыми рисками и возможностями.
  - Описание подхода к постановке и актуализации целей, планов в части использования углеродных единиц для достижения целей.
