# Описание изображения

На изображении представлен фрагмент ESG-отчета, содержащий список аббревиатур с их расшифровками на русском и английском языках. 

## Основные элементы:

### Русская сторона:
- **H23** — Норматив использования собственных средств.
- **ПДН** — Показатель долговой нагрузки.
- **ПИИ** — Персональная идентификационная информация.
- **ПИФ** — Паевой инвестиционный фонд.
- **По** — Программное обеспечение.
- **СБП** — Система быстрых платежей.
- **СХД** — Система хранения данных.

### Английская сторона:
- **AI** — Искусственный интеллект (Artificial Intelligence).
- **API** — Программный интерфейс приложения (Application Programming Interface).
- **B2B** — Продажа «бизнес-бизнесу» (Business-to-Business).
- **B2C** — Бизнес для потребителя (Business-to-Consumer).
- **CAC** — Стоимость привлечения клиентов (Customer Acquisition Cost).
- **CRM** — Система управления взаимоотношениями с клиентами (Customer Relationship Management).

### Визуальные элементы:
- Текст представлен в двух колонках: левая для русского языка, правая для английского.
- Верхняя часть страницы содержит логотип СБЕР и название документа, а также навигационные элементы.

## Заключение
Изображение служит справочным материалом для понимания терминологии, используемой в отчете, что может быть полезно для аналитиков, исследователей и специалистов в области финансов и технологий.
