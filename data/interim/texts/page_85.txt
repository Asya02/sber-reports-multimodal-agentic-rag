# Ключевая информация из изображения

## Раздел: ИТ как бизнес

### Технологическая платформа Platform V
- **Количество инструментов и компонентов**: 60.
- **Основные позиции в российском реестре ПО**:
  - SberLinux OS Server
  - Works
  - Pangolin и др.

### Platform V SberLinux OS Server
- **Описание**: Операционная система для управления корпоративными серверами и обеспечения безопасности выполнения рабочих нагрузок.
- **Основные функции**:
  - Сетевые приложения
  - Базы данных
  - Распределенные вычисления
  - Контейнеризация и виртуализация
  - Искусственный интеллект

### Заключенные коммерческие контракты
- **CNNews**: Награда «Платформа года для импортозамещения».
- **Tadviser IT Prize 2023**: Награда «Платформа года для разработки приложений».

### Сервис GitVerse
- **Объявление на конференции SmartDev**: Анонс сервиса GitVerse для разработки проектов с открытым и закрытым кодом.
- **Возможности GitVerse**:
  - Автоматизация разработки
  - Управление проектами
  - Создание и удаление репозиториев
  - Назначение задач и добавление комментариев
- **Интеграция с AI-помощником GigaCode**: Ускорение написания кода.

### Доступность сервиса
- **С начала 2024 года**: Сервис стал доступен для всех желающих.
