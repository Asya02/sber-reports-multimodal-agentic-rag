# Ключевая информация из документа

## Заголовок
Развитие системы корпоративного управления Сбербанка

## Основные мероприятия 2023 года
- **Обеспечение прав акционеров:**
  - Создание веб-версии приложения для регистрации акционеров.
  - Проведение онлайн-программ для акционеров и работников банка.
  - Информационное взаимодействие через платформу GigaChat.

- **Внедрение лучших практик:**
  - Участие членов Наблюдательного совета в онлайн-программах.
  - Использование искусственного интеллекта для улучшения работы членов НС.
  - Разработка нового функционального приложения для членов НС.

## Направления совершенствования корпоративного управления на 2024 год
- Реализация процедуры выплат невостребованных дивидендов.
- Адаптация GigaChat для акционеров.
- Регулярное проведение мероприятий для участников сообщества.

## Наблюдательный совет и его комитеты
- Создание веб-версии приложения для членов HC SBoard.
- Развитие технологических решений для НС.
- Обсуждение внешних мероприятий по вопросам ESG и развития корпоративного управления.

## Дополнительные замечания
- Важность постоянного взаимодействия с акционерами и их информирования о текущих событиях.
- Применение новых технологий для повышения эффективности корпоративного управления.
