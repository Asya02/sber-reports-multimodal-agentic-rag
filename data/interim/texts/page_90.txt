# Ключевая информация из изображения

## Заголовок
**Развитие искусственного интеллекта**  
**МОДЕЛИ ГЕНЕРАТИВНОГО ИИ**

## Основное содержание
Изображение представляет собой обзор нескольких моделей генеративного искусственного интеллекта, разработанных в рамках стратегии ИИ с 2021 по 2023 годы. Каждая модель описана в отдельном блоке. 

### Модели генеративного ИИ:

1. **GigaChat**
   - Большая языковая модель, взаимодействующая с пользователем в формате диалога.
   - Умеет создавать тексты, писать код и рисовать.
   - Успешно сдала экзамен ЕГЭ по обществознанию на 67 баллов.

2. **GigaCode**
   - ИИ-ассистент разработчика, ускоряющий процесс написания кода.
   - Предлагает наиболее вероятные и релевантные варианты предложений кода в режиме реального времени.

3. **Kandinsky 3.0**
   - Модель, генерирующая высококачественные изображения по текстовому описанию.
   - Способна смешивать картинки и изменять их.
   - Поддерживает 101 язык, включая русский и английский.

4. **Kandinsky Video**
   - Первая российская модель для создания видеороликов по текстовому описанию.
   - Генерирует видео с разрешением 512x512 пикселей и частотой 30 кадров в секунду.

5. **OmniFusion совместно с AIRI**
   - Мультимодальная модель, способная воспринимать разные типы данных.
   - Умеет работать с текстом и изображениями, обеспечивая непрерывный диалог с пользователем.

## Примеры применения ИИ в Сбер
На базе GigaChat запущен ряд ИИ-проектов, включая:
- Использование языковой модели для ускорения ответов на запросы граждан в Московской области.
- Модель для выбора оптимальной локации для открытия новых офисов.
- Модели для оценки кредитного риска на основе новейших архитектур нейронных сетей.
